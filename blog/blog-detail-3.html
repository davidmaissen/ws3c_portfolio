3D effect in just a few clicks

Tags: Photoshop Effects
The term anaglyph process is probably unknown to many, but we all know the 3D glasses with the color filters and the images where the channels are somehow shifted. If you look at such a picture with the 3D color filter glasses, you get a 3D effect. Without the glasses, however, an interesting complementary effect is still created on the image. In Photoshop you can imitate this effect in a short time using the blending options. To do this, we select our source image and copy the layer twice in total. We label the first layer Cyan, open the fill options by double-clicking and select the channels Green and Blue under the advanced fill options. We do the same with the second layer, except that we label it Red and select only the red channel. Finally, we move the individual layers with the Move tool to make the effect visible.
